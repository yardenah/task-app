# Task Manager (Next.js + React)

This project is a simple task management application built using **Next.js (App Router)** and **React**.

The purpose of this project is **learning and practicing Next.js and React concepts**, including:
- Components and props
- State management with `useState`
- Data fetching with `useEffect`
- API routes (GET, POST, PATCH, DELETE)
- Client-side rendering (CSR)
- Routing and layouts in Next.js

---

## 🚀 Features

- Add tasks
- Toggle task completion
- Delete tasks
- Filter tasks (All / Active / Completed)
- Basic full-stack setup using Next.js API routes

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React
- TypeScript
- pnpm
